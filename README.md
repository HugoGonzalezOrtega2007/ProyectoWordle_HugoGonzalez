INSTRUCCIONES DE USO

El juego consta de 10 partidas.

2. Cada partida tiene una palabra secreta de longitud variable.

3. Tienes 5 intentos para adivinar cada palabra.

4. Después de cada intento, se mostrará un resultado usando símbolos:

✅ : Letra correcta en la posición correcta.

⚠️ : Letra en la palabra, pero en otra posición.

❌ : Letra no está en la palabra.

5. Para hacer un intento, escribe una palabra de la misma longitud que la palabra secreta y presiona Enter.

6. Si la palabra no tiene la longitud correcta, el juego pedirá que la escribas nuevamente.

7. Al final de las 10 partidas, se mostrará cuántas palabras acertaste.
